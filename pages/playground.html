Keybindings quick reference

Prefer the keyboard? These shortcuts mirror the ones available in the main app and `/collab.html`:

Ctrl + z
Undo the last change.
Ctrl + y
Redo in shared history.
h / j / k / l
Move selection to parent, next, previous, or child element (also works with arrow keys).
Escape
Clear the current selection, then press again to restore it.
t
Change text content of selected element.
c
Copy the focused element(s).
p / P / o / O
Paste copied elements around or inside current selection.
